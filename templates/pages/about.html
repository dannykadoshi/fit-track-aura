FitTrack Aura is a comprehensive fitness tracking web application designed to help users monitor their workout routines, set and achieve fitness goals, and maintain motivation through visual progress tracking and gamification.

This project was developed as Portfolio Project 4 for the Code Institute Full Stack Software Development Diploma, demonstrating proficiency in full-stack web development using Django, Python, PostgreSQL, and modern web technologies.

The application features a sleek, modern interface with a purple-gradient design theme and dark/light mode support, providing users with an engaging and intuitive experience for tracking their fitness journey across all devices.
